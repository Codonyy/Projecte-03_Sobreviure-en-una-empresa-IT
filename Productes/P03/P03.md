
# P03: Rèplica Web

## Breu descripció

Creació d'una rèplica local d'una web professional desenvolupada amb WordPress per analitzar-ne l'estructura, el disseny, la navegació i l'arquitectura de continguts.

L'objectiu és reproduir fidelment el lloc web original utilitzant **WP Local**, configurant les pàgines, menús, estils i continguts principals per comprendre les bones pràctiques de desenvolupament i manteniment web professional.

Durant el projecte s'investigarà la tecnologia utilitzada pel web original, incloent-hi temes, plugins, tipografies, colors i estructura SEO, amb la finalitat de recrear una experiència d'usuari similar en un entorn local de proves.

---

## Objectius

* Analitzar una web professional creada amb WordPress.
* Reproduir la seva estructura de pàgines i navegació.
* Configurar correctament un lloc WordPress en local.
* Comprendre la importància de la coherència visual i l'experiència d'usuari (UX).
* Desenvolupar habilitats de manteniment i administració de webs corporatives.

---

## Contingut

* Instal·lació del projecte a WP Local.
* Anàlisi de l'estructura del web original.
* Creació de pàgines i menús.
* Configuració del tema i dels plugins necessaris.
* Captures de pantalla del procés.
* Evidències del resultat final.

---

## Resultat final

Una rèplica funcional del web original executada en un entorn local WordPress, mantenint la mateixa estructura de continguts, navegació i aparença visual general.
